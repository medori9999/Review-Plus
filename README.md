# 서론
현재 대한민국에는 제대로된 맛집 추천 시스템이 없다

많이들 이용하는 카x오,네x버 지도 같은 경우 음식점들에 대한 별점 평가가 가능하지만

"광고성 리뷰","무지성 별점"과 같은 문제점들이 있어 음식점에 대한 정확한 평가가 불가능하다,

 

이를 해결하기 위하여 기획하였다

 

제목:Review-Plus(가칭) #0.05퍼

- 당신의 미각을 증명하라

 

핵심 매커니즘

 

1.'신뢰도 점수' 시스템

- 모든 유저는 신뢰도 점수 를 가진다

- 유저 A 가 식당에 6점을 부여한다.

- 이후 유저 B, C, D가 방문하여 3,7,6 (평균6.n) 의 점수를 준다

- A의 보상  : 대중(또는 신뢰도 높은 다른 유저)의 평가와 본인의 평가가 일치함으로 A의 신뢰도 점수 상승

- A의 패널티 : 만약 A가 10점을 줬다면 A는'알바' 거나 '미각이 보편적이지 않음'으로 간주되어 신뢰도 점수하락

- 또한 채점자에게 점수 부여 가능

- 점수에 대한 세부 사항은 개발과정에서 수정예정

2. 랭킹 및 티어 시스템

-단순히 점수만 오르는 것이 아니라, 등급을 나눔(브론즈 실버 골드 ,,,~)

- 랭킹 시스템에 따른 필터 가능 (플레티넘 티어 이상들의 점수) (챌린저들의 인증맛집)

 

고려해봐야 할 사항

a.취향별 시스템 -알고리즘 탑재 느낌

b.어뷰징 - 개인번호 인증 ,영수증 인증 같은시스템 탑재 가능

c.랭킹시스템에 대한 완벽한 고찰
